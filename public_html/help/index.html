Dashboard

This page displays your tasks and tasks shared with you. It includes two tabs: My Tasks and Shared with Me.

Instructions:

  1. Use the tabs at the top to switch between:
    • My Tasks: Displays all tasks created by you.
    • Shared with Me: Displays tasks shared with you by other users.
  2. Click the "Create New Task" button to add a task (see task creation instructions below).
  3. Click on a task in the "My Tasks" tab to view or edit it.
  4. Click on a task in the "Shared with Me" tab to view its details.
  5. Use the "Logout" button in the header to log out of your account.
My Tasks

This tab allows you to view and manage tasks that you have created.

Instructions:

  1. On the Dashboard, navigate to the "My Tasks" tab.
  2. Click "Details" button on a task to open its details.
  3. The Task Details page includes:
    • An Update Task Form: Modify task details such as:
      • Title: Update the task title.
      • Description: Update the task description.
      • Priority: Set the task priority (Low, Medium, or High).
      • Completion: Mark the task as "Completed" or "In Progress."
      • Due Date: Change the due date of the task.
    • A Delete Button: Remove the task permanently. A confirmation popup will appear stating that this action cannot be undone.
    • A Share Button: Share the task with another user. Clicking this button opens a popup where you can select a user from the list of all users in the database.
    • A Shared With Section: Lists all users the task is shared with.
    • An Activity Section: Logs all actions performed on the task. The activity table includes:
      • User: The user who performed the action.
      • Action: The action performed (e.g., created, updated, shared, unshared, completed).
      • Details: Additional information about the action.
      • Date: The date and time the action was performed.
Tasks Shared with Me

This tab allows you to view and edit tasks that have been shared with you. However, you cannot share these tasks with others unless you are the owner.

Instructions:

  1. On the Dashboard, navigate to the "Shared with Me" tab.
  2. Click "Details" button on a shared task to open its details.
  3. The Task Details page includes:
    • An Update Task Form: You can modify task details such as:
      • Title: Update the task title.
      • Description: Update the task description.
      • Priority: Set the task priority (Low, Medium, or High).
      • Completion: Mark the task as "Completed" or "In Progress."
      • Due Date: Change the due date of the task.
    • A Shared With Section: View all users with whom the task is currently shared. You cannot share this task if you are not the owner.
    • An Activity Section: Logs all actions performed on the task. The activity table includes:
      • User: The user who performed the action.
      • Action: The action performed (e.g., created, updated, shared, unshared, completed).
      • Details: Additional information about the action.
      • Date: The date and time the action was performed.
Create Task

This feature allows users to add new tasks to their to-do list.

Instructions:

  1. Click the "Create New Task" button on the Dashboard.
  2. Fill in the task details:
    • Title: The name of the task (required).
    • Description: Additional details about the task (optional).
    • Priority: Set the task priority as Low, Medium, or High.
    • Due Date: Choose the date by which the task should be completed.
  3. Click the "Create" button to add the task to your list.
  4. A popup will appear confirming the task creation. From here, you can:
    • Create another task immediately.
    • Click "Go back" to go to the "My Tasks" tab to view the newly created task.
Task Details

This page allows users to view, update, delete, or share a task. It also provides information about who the task is shared with and logs all activity related to the task.

Instructions:

  1. Click "Details" button on a task in the "My Tasks" or "Shared with me" tab to open its details.
  2. The Task Details page includes:
    • An Update Task Form to modify task details such as title, description, priority, completion and due date.
    • Delete Button: Click this to delete the task. If you are not the owner of the task, it will simply be removed from your shared list. A confirmation popup will appear stating that this action cannot be undone.
    • Share Button: Click this to open a popup where you can select a user from the database to share the task with.
    • A Shared With Section that lists all users the task has been shared with.
    • An Activity Section that logs all actions performed on the task. The activity table includes:
      • User: The user who performed the action.
      • Action: The action performed (e.g., created, updated, shared, unshared, deleted).
      • Details: Additional information about the action.
      • Date: The date and time the action was performed.
Logout

This feature allows users to log out of their account.

Instructions:

  1. On any page inside the Dashboard, click the "Logout" button in the header.
  2. This will clear your session and redirect you to the Authentication Page.
